    Five Common Words

    In this tutorial we will write several morse tests that generate 5 common random words - play them at our desired speed - and test our morse copying accuracy.

    Test by word

    First you will be tested against five generated words on a word-by-word basis. The words that you hear as morse will only be displayed on completion of the word (don’t forget to type a space at the end). Create the CW script five-common-words.rb (shown below) in a text editor, or download via the link and save as five_common_words.rb:

      # five-common-words.rb

require 'cw'

cw do
  comment "5 common words at 12 words per minute"
  shuffle
  word_count 5
  wpm 12
end
                five_common_words.rb    


    Run the script

    To run the test with the following:

    
cw five_common_words

    Test output

    Note: The words in red were entered incorrectly!

    Details of how to adjust colours are available here.

    Test by letter

    Instead of testing word-by-word, you may also test letter-by-letter. Simply add the command test_letters into the CW script (see below) and now you will be marked letter-by-letter rather than word-by-word.

    Tip: You may find this kind of test more informative if you are frequently guessing some letters incorrectly.
      # five_common_words_2.rb

require 'cw'

cw do
  comment "5 common words at 12 words per minute"
  shuffle
  wpm        12
  word_count 5
  test_letters    # add the command test_letters here
end
                five_common_words_2.rb    


    Result:

    Note: The letters in red were entered incorrectly!

    Combine both examples

    Here we will combine both examples into a single script. In practice you may end up having CW scripts comprising of many tests (perhaps a script called daily.rb or morning.rb). Create or download the combined CW script (or download it) and run it.

      # five_common_words_3.rb

require 'cw'

cw do
  comment "5 common words at 12 wpm (test letters)"
  shuffle
  wpm        12
  word_count 5
  test_letters
end

cw do
  comment "5 common words at 12 wpm (test words)"
  shuffle
  wpm        12
  word_count 5
end
                five_common_words_3.rb    


    Repeat until correct

    Replace the print_letters command with the repeat_word command.

    Tip: The repeat_word command will cause CW to replay the current word if it has been entered incorrectly, (or hasn’t been entered at all).
      # five-common-words-4.rb

require 'cw'

cw do
  comment "5 common words at 12 wpm (test letters)"
  shuffle
  wpm        12
  word_count 5
  repeat_word
end
                five_common_words_4.rb    


    Example with just word_count

    Remove all commands apart from word_count and see what happens.

      # five_common_words_5.rb

require 'cw'

cw do
  word_count 5
end
                five_common_words_5.rb    


    Note: Default values are used for WPM, and since shuffle is not declared, the word order will be repeated on subsequent runs.
    Important: Warning! Leaving out word_count (or one of its aliases) will use the entire dictionary! You can quit the test by entering four “q” keystrokes in a row should you happen to try this!
